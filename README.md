# Проект магазина
Онлайн-магазин контента под самые различные задачи

## Описание проекта
С помощью HTML, CSS, JS создан многостраничный сайт магазина. 
Верстка сайта адаптивна (подходит как для больших экранов, так и для смартфонов) и доступна (адекватное изучение сайта скринридерами: используются подходящие, не универсальные, блоки, альтернативы для изображений, структура страницы).  
[Макет](https://www.figma.com/design/P0YTj9wOh27KOI69ighVEf/AddMarkets--?node-id=2822-10917&t=Ld5jKvMpMdD6IhNJ-0) взят из открытых источников

## Структура проекта
1. `index.html` — главная программы
2. `styles/*.css` — стили страниц и переменные
3. `fonts/*.css` — шрифты, используемые на сайте
3. `scripts/script.js` — анимации (в основном загрузочные)
